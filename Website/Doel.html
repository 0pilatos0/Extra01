Waarop wil ik me graag ontwikkelen bij deze opleiding?

Ik wil me graag ontwikkelen op elk mogelijk opzicht met alles wat vanpas komt bij het uitvoeren van een baan als programmeur en zo dus de sociale vaardigheden en het verstand van het programmeren en de mogelijkheden om snel researh te kunnen doen. Wat ook een belangrijk punt is dat ik me wil ontwikkellen op de code zelf, om de code zo compact mogelijk te schijven en zo weinig mogelijk fouten te maken in mijn code die ik allemaal nog ga schijven
